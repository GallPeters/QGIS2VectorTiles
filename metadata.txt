about=QGIS2VectorTiles turns your QGIS project into a self-contained, ready-to-share web map package in a single step. It exports a lightweight MBTiles vector tile source, a client-side MapLibre JSON style faithfully derived from your QGIS symbology (including data-defined properties), a QLR layer for seamless reuse inside QGIS, and an offline-friendly HTML viewer for instant preview and sharing. Built for cartographers and GIS professionals who need a streamlined desktop-to-web workflow without sacrificing styling fidelity.
