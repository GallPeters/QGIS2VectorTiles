## Output Package

- Vector tiles source (`XYZ` directory)
- Styled QGIS layer file (`.qlr`)
- MapLibre style (`.json`)
- MapLibre sprites *(optional)* (`.png` + `.json`)
- A ready-to-use MapLibre viewer (`.html`)
