Execution Pipeline

Core
QGIS2VectorTiles.py Execution wrapper and environment manager
1
rules_flattener.py Flattens complex nested symbology and filters
2
rules_exporter.py Trims bounds and exports to intermediate vectors
3
tiles_generator.py Slices geometries via multi-threaded GDAL
4
tiles_styler.py Binds original QGIS styles to vector channels
5
maplibre_converter.py Translates configs to MapLibre JSON schema
6
server_initializer.py Prepares ready-to-deploy web environments

Applications

Versatile Deployments

Built to fit seamlessly into modern spatial stacks.

01

Client-Side Rendering

Drop the MapLibre style into your web map client. No server processing or on-the-fly rasterization-just fast, sharp, fully styled vector tiles rendered natively.

Mapbox MapTiler MapLibre OpenLayers

Compatible with Vector Tiles and GL Styles.

02

Web Map Services

Serve vector tiles through standard GIS infrastructure. Benefit from smaller payloads, faster response times, and a single source of truth for your styles.

GeoServer QGIS Server

Using MBTiles and MBStyle extensions.

03

Project Sharing

Share a single styled layer file instead of dozens of datasets, projections, and dependencies. The complex multi-source project is collapsed with visuals baked in.
